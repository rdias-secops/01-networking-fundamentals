# Networking Fundamentals

This repository contains practical networking studies focused on cybersecurity fundamentals using Linux, Windows, Wireshark, and networking tools.

## Topics Covered

- IP Address
- Subnet Mask
- Default Gateway
- DNS
- TCP/IP
- Network Ports

## Technologies Used

- Kali Linux
- Ubuntu
- Windows
- Wireshark
- VirtualBox

## Practical Activities

- Packet analysis
- TCP stream inspection
- Port inspection
- Local HTTP server creation
- TCP vs UDP analysis
- DNS resolution analysis
