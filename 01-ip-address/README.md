# IP Address

## Objective

Understand how IP addresses work and how to identify network information on Windows and Linux systems.

---

## Windows

### Command Used

```bash
ipconfig
```

### Description

The 'ipconfig' command displays network configuration information on Windows systems.


![Windows IPConfig](images/ip-address%20windows.jpeg)


### Information Identified

- IPv4 Address
- Subnet Mask
- Default Gateway
- DNS Server

---

## Linux

### Command Used

```bash
ip a
```

### Description

The 'ip a' command displays IP address informationand network interfaces on Linux systems.


![Linux IP A](images/ip-address%20linux.jpeg)


### Information Identified

- Interface name
- IPv4 address (inet)
- Loopback address
- MAC address
- CIDR notation
